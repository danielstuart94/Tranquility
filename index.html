Real‑time comfort scores for public spaces

Simple. Human. One number that tells you when and where to sit outside. Optimised for UK realities and Mediterranean dreams.

Try the Lyric Square demo   View on GitHub

How it works

Weather + wind + air + noise + sun/shade + busyness → comfort score 0–100. Rain is scaled by shelter.

Why it matters

Better lunch breaks, smarter event timing, higher footfall. A small nudge to use communal spaces more.

For partners

Councils, BIDs, landlords: pilot per site, sponsor the QR page, access an API for dashboards.

Scan‑to‑use

Place a QR in your square that opens the live score. Users can give one‑tap feedback.

Privacy first

We collect only anonymous slider states and feedback with consent. No cookies beyond local settings.
